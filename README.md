### SRX

SRX（Segmentation Rules eXchange）是专门用于表示句段分割规则的格式。

将正则表达式作为规则进行存储，而且处理的先后顺序很重要，另外它也有分例外规则和分割标准。

分割规则，它有分在分割前出现的内容（beforebreak）和分割后出现的内容（afterbreak）
